Propiedad outline en CSS

1) ¿Qué es outline?

outline es una línea que se dibuja por fuera del borde (border) de un elemento. A diferencia de border, no ocupa espacio ni afecta al flujo de la página.

  • Se usa mucho para enfocar elementos (focus), especialmente en accesibilidad.
  • Puede tener color, grosor, estilo y también outline-offset para separar la línea del elemento.

2) Sintaxis

outline: <grosor> <estilo> <color>;

/* Ejemplo */
outline: 3px solid red;

outline-color: red;
outline-style: dashed;
outline-width: 4px;

outline-offset: 6px; /* separa la línea del borde */

3) Ejemplos visuales

Solid
outline: 2px solid verde
Dotted
outline: 3px dotted azul
Dashed
outline: 3px dashed rosa
Double
outline: 5px double naranja
Offset
outline-offset: 8px

4) Usos en accesibilidad

Cuando navegamos con teclado, el navegador dibuja un outline automático para indicar el elemento activo.

button:focus{
  outline: 3px solid #0d6efd;
  outline-offset: 4px;
}

No lo elimines sin dar una alternativa visual, ya que es clave para usuarios que navegan con teclado.

5) Ejemplo de quitar outline (no recomendado sin reemplazo)

.outline-none:focus{
  outline: none; /* cuidado! */
  border: 2px solid red; /* reemplazo visual */
}

6) Resumen rápido

  • outline = línea por fuera del borde, no ocupa espacio.
  • No es lo mismo que border (que sí afecta el tamaño).
  • outline-offset separa el outline del elemento.
  • Importante para accesibilidad (enfoque con teclado).
