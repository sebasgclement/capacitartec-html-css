Emmet – Escribir HTML y CSS a la velocidad de la luz ⚡

¿Qué es Emmet?

Emmet es un sistema de abreviaturas que expanden a código HTML/CSS completo en tu editor (VS Code lo trae incorporado). Escribís algo corto, apretás Tab y ¡boom!, tenés la estructura lista.

VS Code: Funciona por defecto. Si no expande con Tab, verificá en Settings → Emmet: Trigger Expansion On Tab y que el lenguaje del archivo sea el correcto (HTML, CSS, etc.).

Conceptos clave (sintaxis)

  • Multiplicación li*3 → repite elementos.
  • Hijos ul>li | Hermanos h1+p | Subir nivel section>h2^p.
  • Clases .card | ID #hero | Atributos a[href="/"].
  • Texto a{Click aquí}.
  • Numeración li.item$*5 → .item1, .item2… | $$ agrega ceros: 01, 02.
  • Agrupar (header>nav>ul>li*3)>main+footer.
  • Lipsum lorem10 → 10 palabras de texto de prueba.
  • Atajos HTML5 ! → boilerplate; link:css, script:src, etc.

HTML: ejemplos de uso

Navbar simple

nav>ul.nav>li*3>a{Item $}

Expande a:

<nav>
  <ul class="nav">
    <li><a href="">Item 1</a></li>
    <li><a href="">Item 2</a></li>
    <li><a href="">Item 3</a></li>
  </ul>
</nav>

Card con imagen y CTA

.card>img.card__img[src="img.jpg" alt=""]+h3.card__title{Título}+p.card__desc{lorem15}+a.btn[href="#"]{Comprar}

Formulario básico

form.form>(label[for=name]{Nombre}+input#name[type=text])+ (label[for=mail]{Email}+input#mail[type=email])+button.btn{Enviar}

Tabla rápida

table.table>(thead>tr>th*3{Col $})+(tbody>tr*3>td*3{Dato})

Layout base

header#top.header>h1{Sitio}+nav>ul>li*4>a{Enlace $}^^main>section.section*2>h2{Sección $}+p{lorem20}^^footer.footer{© 2025}

Links e imágenes

a[href="https://example.com" target=_blank]{Visitar} + img[alt="Paisaje" src="img/paisaje.jpg"]

CSS: abreviaturas útiles

En archivos CSS/SCSS Emmet expande propiedades completas:

Box model

m10        → margin: 10px;
mt8        → margin-top: 8px;
p10-20     → padding: 10px 20px;
w100       → width: 100px;
h100p      → height: 100%;

Display & Flex

d:f        → display: flex;
fd:c       → flex-direction: column;
ai:c       → align-items: center;
jc:sb      → justify-content: space-between;
g10        → gap: 10px;

Posicionamiento

pos:a      → position: absolute;
t0 l0      → top: 0; left: 0;
z10        → z-index: 10;

Tipografía

ff:sans    → font-family: sans-serif;
fw700      → font-weight: 700;
fs14       → font-size: 14px;
ta:c       → text-align: center;

Colores & fondo

c#333      → color: #333;
bg#f5f5f5  → background: #f5f5f5;
bgc#000.5  → background-color: rgba(0,0,0,.5);

Bordes & sombras

bd1s#ccc   → border: 1px solid #ccc;
bdrs8      → border-radius: 8px;
bxsh       → box-shadow: 0 0 0 #000; (luego ajustás valores)
Tip: Si una abreviatura no expande, escribí la propiedad corta (ej. bgc) y apretá Tab. Emmet sugiere variaciones en el Intellisense.

Atajos “estrella” para usar todos los días

Estructura HTML5

!

Boilerplate completo (doctype, head, meta, body).

Lista con texto

ul>li.item$*5{Elemento $}

Grilla de cards

.grid>.card*3>img.card__img[src="img$.jpg"]+h3.card__title{Card $}+p.card__txt{lorem10}

Formulario express

form>(label+input:text+br)*2+button{Enviar}

Flujo de trabajo recomendado

  1. Escribí la abreviatura (pensá en estructura, no en etiquetas sueltas).
  2. Presioná Tab para expandir.
  3. Movete con Tab por los “cursos” (lugares editables) y completá textos/attrs.
  4. Usá Wrap with Abbreviation (⌘/Ctrl+Shift+P) para envolver contenido existente.

Mini ejercicio

Escribí y expandí esta abreviatura para una landing simple:

header.header>h1.logo{Marca}+nav>ul.menu>li*4>a[href="#"]{Sección $}^^main>section.hero>.hero__content>h2{Título destacado}+p{lorem20}+a.btn[href="#"]{Empezar}^^footer.footer>p{© 2025}

Resumen machetero

  • > hijo | + hermano | ^ subir | * repetir | {} texto | [] atributos.
  • . clase | # id | $ numeración.
  • ! HTML5 rápido | loremN texto muestra | link:css, script:src.
  • En CSS, abreviaturas como d:f, ai:c, jc:c, m10, p10-20, bd1s#ccc, bdrs8, c#333, bg#eee.
