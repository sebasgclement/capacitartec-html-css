Control de flujo de texto (CSS)

1) ¿Qué es?

Es el conjunto de técnicas para decidir cómo envuelve, corta o guiona el texto dentro de un contenedor. Con esto evitás desbordes feos, mejorás legibilidad y controlás alineaciones en tarjetas, tablas o layouts responsivos.

  • white-space: cómo se tratan los espacios y saltos de línea.
  • overflow-wrap / word-wrap: cómo romper palabras largas.
  • word-break: estrategia de corte dentro de palabras.
  • hyphens: guionado automático o manual.
  • text-overflow: puntos suspensivos en una línea.
  • line-clamp: recortar a N líneas con “…” (multilínea).

2) white-space

Controla espacios/saltos: normal, nowrap, pre, pre-wrap, pre-line.

white-space: normal;
Texto con espacios múltiples y saltos de línea.
white-space: nowrap;
Texto con espacios que NO se corta en varias líneas (puede desbordar si no hay ancho).
white-space: pre;
Texto con espacios y saltos de línea preservados.
white-space: pre-wrap;
Texto con espacios preservados y también permite envolver líneas.

3) Cortes de palabras largas

Útil para URLs, códigos o identificadores infinitos que rompen el layout.

overflow-wrap: normal;
PalabraLarguísimaSinPuntosDeCorte_123456789ABCDEFGHIJK
overflow-wrap: break-word;
PalabraLarguísimaSinPuntosDeCorte_123456789ABCDEFGHIJK
overflow-wrap: anywhere;
PalabraLarguísimaSinPuntosDeCorte_123456789ABCDEFGHIJK

Alternativa: word-break: break-all; (rompe más agresivo, puede cortar palabras normales).

4) word-break

word-break: normal;
Supercalifragilisticoespialidoso en varias líneas si hace falta.
word-break: break-all;
Supercalifragilisticoespialidoso rompe en cualquier carácter si es necesario.
word-break: keep-all;
Texto continuo que intenta evitar cortes dentro de palabras.

5) Guionado con hyphens

Permite cortes con guion (‑). Requiere diccionarios del navegador/idioma correcto.

hyphens: none;
Internacionalización y accesibilidad mejoran la experiencia de usuario en dispositivos móviles.
hyphens: manual; + &shy;
Internacio­nalización y accesibi­lidad con puntos de corte manuales.
hyphens: auto;
Internacionalización y accesibilidad mejoran la experiencia de usuario en dispositivos móviles.

6) Puntos suspensivos (una línea)

text-overflow: ellipsis; necesita ancho limitado, overflow:hidden y normalmente white-space: nowrap.

.ellipsis (corta en 1 línea)
Este es un texto muy largo que no entra en una sola línea y queremos que termine con puntos suspensivos… 
.title{
  max-width: 420px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

7) Clamp (N líneas)

Para tarjetas/noticias: recorta a 3 líneas y agrega “…” implícitos.

-webkit-line-clamp: 3;
CSS nos permite controlar la cantidad de líneas visibles de un párrafo largo con line-clamp. Es ideal para tarjetas en grid, listados de noticias o catálogos, evitando que un texto excesivo rompa el layout y manteniendo consistencia visual. 
.excerpt{
  display:-webkit-box;
  -webkit-line-clamp:3;
  -webkit-box-orient:vertical;
  overflow:hidden;
}

8) Mini ejercicio

Convertí las descripciones de una card en máximo 2 líneas con clamp y aplicá overflow-wrap: break-word para URLs:

.card__desc{
  display:-webkit-box;
  -webkit-line-clamp:2;
  -webkit-box-orient:vertical;
  overflow:hidden;
  overflow-wrap: break-word;
}

9) Resumen rápido

  • white-space controla espacios y saltos (nowrap evita cortes).
  • overflow-wrap / break-word rompen palabras largas sin arruinar el layout.
  • word-break: break-all es agresivo: usalo con cuidado.
  • hyphens: auto guiona palabras (según idioma).
  • text-overflow: ellipsis = “…” en una línea.
  • line-clamp corta a N líneas (ideal para tarjetas y listas).
