Desventajas sin media queries:
  • Sidebars rígidas: con flex-basis fijo (p. ej. 260px) en pantallas chicas invaden el contenido o generan scroll horizontal.
  • Texto aplastado: columnas con flex:1 1 600px se encogen demasiado; la lectura se vuelve incómoda.
  • Orden visual ≠ semántico: si reordenás con order para móvil/escritorio, sin media queries podés confundir usuarios/lectores de pantalla.
  • Alturas impredecibles: al wrappear tarjetas, algunas quedan largas/cortas y el flujo no siempre luce “cuadriculado”.
  • Componentes mezclados: un layout complejo suele requerir cambiar de 3 columnas → 1 columna; sin @media no tenés ese quiebre.

Recomendación: combinar Flexbox con @media para redefinir flex-basis, order, gap y orientación (row/column) según ancho de pantalla. Para layouts generales, CSS Grid suele ser más cómodo.
