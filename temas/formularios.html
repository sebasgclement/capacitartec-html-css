Formularios en HTML

Los formularios permiten ingresar y enviar datos. Cada elemento tiene atributos que modifican su comportamiento. A continuación, mostramos ejemplos interactivos con sus atributos explicados.

Formulario básico

<input type="text">

  • type: especifica el tipo de dato (text, email, number, etc.)
  • placeholder: muestra un texto guía dentro del campo
  • name: nombre que se enviará en el formulario
  • required: hace obligatorio completar el campo

<input type="email">

  • Valida que se ingrese un correo electrónico válido.

<input type="number" min="1" max="120">

  • min y max: definen los valores permitidos

<input type="date">

  • Muestra un calendario para elegir fechas.

<textarea rows="4"></textarea>

  • Permite ingresar texto largo. rows define la altura.

Opciones y selección

<select> + <option>

  • Menú desplegable de opciones.
  • value: lo que se envía del campo seleccionado.

<input type="radio">

  • Permite elegir solo una opción del grupo (mismo name).



<input type="checkbox">

  • Permite seleccionar varias opciones.
