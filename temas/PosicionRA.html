Posición Relativa y Absoluta

1. ¿Qué es position?

La propiedad position en CSS define cómo se ubica un elemento en la página. Los valores más comunes son relative y absolute.

2. Position Relative (relative)

  • El elemento mantiene su espacio original en el flujo del documento.
  • Se puede mover usando top, right, bottom, left en relación a su posición original.
  • Sirve como referencia para elementos con position: absolute dentro de él.
Relativa

3. Position Absolute (absolute)

  • El elemento se posiciona de forma absoluta respecto al ancestro más cercano que tenga position distinto de static.
  • No ocupa espacio en el flujo normal del documento.
  • Si no hay ancestro posicionado, se toma como referencia el body.
Absoluta

5. Mini ejercicio

Crea un contenedor con position: relative y coloca una caja en la esquina inferior derecha usando position: absolute.


.contenedor {
  position: relative;
  width: 300px;
  height: 200px;
  background: lightgray;
}
.caja {
  position: absolute;
  bottom: 10px;
  right: 10px;
}

6. Resumen visual

  • Relative: se mueve desde su posición original y mantiene su espacio.
  • Absolute: se posiciona respecto a un contenedor posicionado y no ocupa espacio.
