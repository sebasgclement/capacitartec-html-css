object-fit y object-position

1) ¿Qué son?

object-fit controla cómo encaja un contenido reemplazado (ej: <img>, <video>) dentro de su contenedor cuando hay diferencia de proporciones.

object-position define desde qué punto se recorta o alinea la imagen/video dentro del contenedor.

  • Valores de object-fit: contain, cover, fill, none, scale-down.
  • Valores de object-position: palabras clave (top, left, center…), o coordenadas (20% 80%, 10px 5px).

Útil en cards, galerías, thumbnails y headers donde el recorte importa.

2) Demos de object-fit

contain — Ajusta sin recortar (puede dejar “bandas”)
cover — Rellena recortando lo que sobre
fill — Estira para ocupar (puede deformar)
none — No escala ni recorta (puede desbordar)
scale-down — Elige entre none o contain, la que deje la imagen más chica

4) También funciona con <video>

object-fit: cover + object-position: center
object-fit: contain

Ideal para headers con video responsivo sin deformar.

5) Código base

/* Imagen/Video responsivo dentro de un marco fijo */
.card__media{
  width:100%; height:100%;
  object-fit: cover;          /* o contain/none/fill/scale-down */
  object-position: center;    /* o top/left/20% 80% etc. */
}
/* El contenedor define la proporción/recorte */
.card__frame{
  aspect-ratio: 16/9; /* o 1/1 para cuadrado, 3/4 vertical */
  overflow:hidden; border-radius:10px;
}

6) Mini ejercicio

Hacé que la imagen de portada de una card se vea siempre completa en móvil sin bandas laterales:

.card__frame{ aspect-ratio: 4/3; overflow:hidden; }
.card__img{
  width:100%; height:100%;
  object-fit: cover;
  object-position: 50% 30%; /* priorizá el sujeto (ojos/rostro) */
}

7) Resumen rápido

  • contain = todo visible (puede dejar banda gris/espacio).
  • cover = llena el marco recortando bordes (lo más usado en cards/headers).
  • fill = deforma para ocupar (evitalo salvo casos puntuales).
  • none = sin escala (puede desbordar).
  • scale-down = elige entre none o contain (la menor).
  • object-position te deja “encuadrar” la parte importante.
