🧠 ¿Qué es metatags.io?

metatags.io es una herramienta online que permite ver cómo se mostrará tu sitio web al compartirlo en redes sociales como Facebook, Twitter, LinkedIn, entre otras.

🔎 ¿Para qué sirve?

  • Simula cómo se ve tu página en redes.
  • Permite modificar título, descripción e imagen.
  • Te genera automáticamente el código HTML listo para usar.

🚀 ¿Cómo usarla?

  1. Entrá a https://metatags.io
  2. Pegá la URL de tu sitio (puede ser temporal como GitHub Pages).
  3. Modificá los campos: título, descripción, imagen.
  4. ¡Listo! Copiá el código generado y pegalo dentro del <head> de tu HTML.

📦 Código de ejemplo generado:

<!-- Meta Tags generados por metatags.io -->
<meta property="og:title" content="Curso HTML y CSS gratuito">
<meta property="og:description" content="Aprendé a crear sitios web desde cero con este curso práctico de HTML y CSS">
<meta property="og:image" content="https://tusitio.com/imagen-del-curso.jpg">
<meta property="og:url" content="https://tusitio.com">

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Curso HTML y CSS gratuito">
<meta name="twitter:description" content="Aprendé a crear sitios web desde cero con este curso práctico de HTML y CSS">
<meta name="twitter:image" content="https://tusitio.com/imagen-del-curso.jpg">
