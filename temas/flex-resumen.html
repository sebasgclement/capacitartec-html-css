Resumen — CSS Flexbox

Cheat‑sheet compacto con propiedades, valores por defecto, atajos y tips.

Conceptos clave

  • Eje principal: definido por flex-direction (por defecto, horizontal izquierda→derecha).
  • Eje cruzado: perpendicular al principal.
  • Unidimensional: Flexbox organiza en fila o columna (para 2D, suele convenir Grid).
  • Shorthand: flex y flex-flow agrupan valores frecuentes.

Truco: margin-left:auto empuja un ítem hacia el extremo opuesto en row.

Ejes (visual)

row → eje principal horizontal column → eje principal vertical justify-* → a lo largo del principal align-* → a lo largo del cruzado

Propiedades del contenedor (padre)

PropiedadValores comunesDefaultNotas
display flex, inline-flex — Activa el contexto flex.
flex-direction row, row-reverse, column, column-reverse row Define el eje principal.
flex-wrap nowrap, wrap, wrap-reverse nowrap Permite múltiples líneas de ítems.
flex-flow <direction> <wrap> (p. ej. row wrap) row nowrap Atajo de flex-direction + flex-wrap.
justify-content flex-start, center, flex-end, space-between, space-around, space-evenly flex-start Distribución a lo largo del eje principal.
align-items stretch, flex-start, center, flex-end, baseline stretch Alineación en el eje cruzado (una sola línea).
align-content stretch, flex-start, center, flex-end, space-between, space-around normal (efecto tipo stretch) Solo aplica cuando hay wrap y múltiples líneas.
gap row-gap, column-gap, p. ej. gap:12px 0 Espacio entre ítems (sí funciona en Flex).
place-content <align-content> / <justify-content> — Atajo para contenido en ambos ejes (multi-línea).
justify-items — — No tiene efecto en contenedores flex.

Propiedades de los ítems (hijos)

PropiedadValoresDefaultNotas
order Entero (negativo/positivo) 0 Cambia el orden visual, no el del DOM (accesibilidad/tabbing).
flex-grow Número ≥ 0 0 Proporción para crecer si sobra espacio.
flex-shrink Número ≥ 0 1 Proporción para encogerse si falta espacio.
flex-basis auto, content, longitudes (px, %, etc.) auto Tamaño base preferido del ítem.
flex <grow> <shrink> <basis> (p. ej. 1 1 200px) 0 1 auto Atajo más usado.
align-self auto, stretch, flex-start, center, flex-end, baseline auto (hereda de align-items) Anula alineación del ítem individual.
min-width/min-height longitudes auto Útiles para evitar “aplastamiento” excesivo.

Valores por defecto más usados

Contenedor: display:flex, flex-direction:row, flex-wrap:nowrap justify-content:flex-start, align-items:stretch, align-content:normal, gap:0 Ítem: order:0, flex:0 1 auto, align-self:auto

Atajos útiles

  • flex:1 ≈ 1 1 0 (crece y se reparte por igual).
  • flex:0 0 260px → ancho fijo tipo sidebar.
  • flex-flow: row wrap → dirección + múltiples líneas en una.
  • place-content: center → atajo de align-content + justify-content (multi-línea).

Gotchas / Errores comunes

  • Usar align-content sin wrap: no tendrá efecto.
  • Reordenar con order puede confundir a lectores de pantalla y el foco de teclado.
  • Texto “aplastado”: agrega min-width/min-height o limita flex-shrink.
  • Querer cuadrículas perfectas con Flex: para layouts 2D, CSS Grid suele ser mejor.

Snippets rápidos

/* Repartir columnas iguales con salto por ancho */
.container { display:flex; flex-wrap:wrap; gap:12px; }
.col { flex:1 1 240px; }

/* Empujar un ítem al extremo derecho en row */
.toolbar { display:flex; }
.spacer { margin-left:auto; }

/* Sidebar fija + contenido flexible */
.layout { display:flex; gap:16px; }
.sidebar { flex:0 0 260px; }
.content { flex:1 1 600px; }

/* Evitar aplastamiento excesivo */
.card { flex:1 1 300px; min-width:280px; }

Cuándo usar Flex vs Grid

  • Flex: barras, cards en fila/columna, alineaciones simples, componentes.
  • Grid: layouts de página, rejillas 2D, alineaciones por filas y columnas a la vez.

Podés combinarlos: Grid para el layout general y Flex dentro de cada sección/componente.
