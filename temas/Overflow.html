Propiedad overflow en CSS

1) ¿Qué es?

overflow controla cómo se maneja el contenido que se desborda de un contenedor cuando es más grande que él.

  • visible: contenido se muestra aunque se salga (por defecto).
  • hidden: el contenido que se sale se recorta y no se ve.
  • scroll: siempre muestra barras de scroll (horizontal y/o vertical).
  • auto: muestra barras de scroll solo si es necesario.

3) Overflow en ejes independientes

Podés controlar cada eje por separado:

  • overflow-x: controla el desborde horizontal.
  • overflow-y: controla el desborde vertical.
/* Solo scroll vertical */
.caja { overflow-y: scroll; overflow-x: hidden; }

4) Buenas prácticas

  • Evitar scroll constante si no es necesario → usar auto.
  • En diseños responsive, el scroll horizontal suele indicar problemas de maquetado.
  • Para ocultar elementos sin alterar el flujo, overflow:hidden + max-height es útil.

5) Resumen rápido

  • visible: deja que el contenido se desborde (default).
  • hidden: recorta lo que se sale.
  • scroll: siempre muestra scroll.
  • auto: scroll solo si hace falta.
  • Podés separar por ejes con overflow-x y overflow-y.
