Selectores Avanzados en CSS

1. Selectores de atributos

Permiten aplicar estilos a elementos HTML en base a sus atributos y valores.

  • [attr]: elementos que tienen el atributo especificado.
  • [attr="valor"]: elementos cuyo atributo tiene exactamente ese valor.
  • [attr^="valor"]: atributo que comienza con el valor indicado.
  • [attr$="valor"]: atributo que termina con el valor indicado.
  • [attr*="valor"]: atributo que contiene el valor indicado.

2. Combinadores

Definen la relación entre elementos HTML para aplicar estilos:

  • E1 > E2: E2 es hijo directo de E1.
  • E1 + E2: E2 es el elemento inmediatamente siguiente a E1.
  • E1 ~ E2: E2 es cualquier hermano posterior de E1.

3. Pseudo-clases estructurales

Seleccionan elementos según su posición o tipo dentro de un contenedor:

  • :first-child: primer hijo.
  • :last-child: último hijo.
  • :nth-child(n): hijo en la posición indicada (o patrón).
  • :nth-of-type(n): elemento del tipo especificado en esa posición.

Ejemplo:

  • Elemento 1 (primer hijo)
  • Elemento 2 (nth-of-type(2))
  • Elemento 3
  • Elemento 4
  • Elemento 5 (último hijo)

4. Pseudo-clases de interacción

Estilos aplicados en respuesta a acciones del usuario:

  • :hover: cuando el cursor pasa por encima.
  • :focus: cuando un elemento recibe foco (ej. input activo).
  • :active: cuando el elemento está siendo clicado.
