Breakpoints

Escala: xs (<576), sm (≥576), md (≥768), lg (≥992), xl (≥1200), xxl (≥1400)

/* Patrón: {prop}-{bp?}-{valor} */
d-none d-md-block
text-center text-lg-start
p-2 p-md-3 p-xl-4

Sistema de grillas (12 columnas)

  • Fila: .row — Columnas: .col / .col-* / .col-{bp}-*
  • Gaps: .g-0….g-5 + variantes .gx-*, .gy-*
  • Offset y orden: .offset-*, .order-*
<div class="row g-3">
  <div class="col-12 col-md-6 col-lg-4">A</div>
  <div class="col-12 col-md-6 col-lg-4">B</div>
  <div class="col-12 col-lg-4">C</div>
</div>
<div class="row">
  <div class="col-4">1</div>
  <div class="col-4 offset-4">2 (offset 4)</div>
</div>

Espaciado

m (margin), p (padding) + lados t/b/s/e/x/y y escala 0–5, auto.

p-3 py-4 px-2 mt-2 ms-auto mb-0
p-md-4 mt-lg-5 mx-auto

Bootstrap + CSS propio

  • Creamos clases propias y las combinamos con utilitarias.
  • Para overrides, cargá tu CSS después del CSS de Bootstrap.
/* custom.css */
.btn-brand{background:#6610f2;color:#fff;border-radius:999px}
.btn-brand:hover{background:#520dc2}

Buenas prácticas

  • Mobile first: usá bp ascendentes (-sm, -md…)
  • No repitas 10 utilitarias → movelas a una clase.
  • Accesibilidad: foco visible, contraste, .visually-hidden
  • Performance: no anidar grillas de más; preferí gap
  • JS: usá el bundle oficial para modales/offcanvas
@media (prefers-reduced-motion: reduce){
  *{scroll-behavior:auto!important}
}
Atajo mental: {prop}-{bp?}-{valor} → d-md-flex, px-3, text-lg-start.
