📚 Resumen - HTML Intermedio

En esta sección profundizamos los conocimientos de HTML, aprendiendo etiquetas más avanzadas, mejores prácticas y conceptos clave para crear sitios accesibles, estructurados y modernos.

🗒️ Comentarios

Sirven para dejar notas en el código que no se muestran en el navegador. Se escriben con <!-- comentario -->.

🌐 Favicon

Se agrega en el <head> usando:

<link rel="icon" href="favicon.ico" type="image/x-icon">

🔖 Metadatos

Etiquetas como:

  • <meta charset="UTF-8">
  • <meta name="viewport">
  • <meta name="description">
  • <title>

🔍 metatags.io

Genera etiquetas como:

  • <meta property="og:title">
  • <meta name="twitter:card">

📝 Textarea

Etiqueta para ingresar texto en varias líneas:

<textarea rows="4" cols="50"></textarea>

🏷️ Label

Vincula texto con inputs:

  • <label for="email">
  • <input id="email">

🔽 Select, Datalist y Option

  • <select> / <option>
  • <input list="..."> / <datalist>

📦 Fieldset y Legend

Se usan para agrupar inputs:

  • <fieldset>
  • <legend>

📂 Details y Summary

Para mostrar/ocultar contenido:

  • <details>
  • <summary>

🔗 Enlaces avanzados

  • <a href="..." target="_blank"> (externo)
  • <a href="#seccion"> (ancla interna)
  • mailto: y tel:

🎧🎥 Audio y Video

  • <audio controls><source></audio>
  • <video controls poster="..."><source></video>

🚀 Lazy Loading

Se agrega a imágenes para optimizar carga:

<img src="..." loading="lazy">

🛑 HTML Obsoleto

Evitar etiquetas como:

  • <font>
  • <center>
  • <marquee>

📐 HTML Semántico

  • <header>, <main>, <footer>
  • <section>, <article>, <aside>

♿ Accesibilidad Web

  • alt en imágenes
  • aria-label, aria-required, role
  • Contrastes, navegación con teclado, etiquetas claras
✨ ¡Felicitaciones! Si llegaste hasta acá, ya dominás las bases y el nivel intermedio de HTML. ¡Estás listo/a para comenzar con CSS intermedio!
