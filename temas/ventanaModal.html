Ventanas Modal

1. ¿Qué es una ventana modal?

Una ventana modal es un elemento superpuesto que aparece sobre el contenido principal, obligando al usuario a interactuar con ella antes de volver a la página. Se usa para mostrar mensajes, formularios o confirmaciones.

2. Partes de un modal

  • Fondo: capa semitransparente que cubre la pantalla (background: rgba()).
  • Contenido: caja central con el mensaje o elementos interactivos.
  • Botón de cierre: permite cerrar la ventana.

4. Código base


/* HTML */
<button id="abrirModal">Abrir Modal</button>
<div class="modal" id="miModal">
  <div class="modal-contenido">
    <h2>Título</h2>
    <p>Contenido del modal</p>
    <button class="cerrar" id="cerrarModal">Cerrar</button>
  </div>
</div>

/* CSS */
.modal { display: none; position: fixed; ... }
.modal-contenido { background: white; padding: 20px; }

/* JS */
document.getElementById("abrirModal").onclick = function() {
  document.getElementById("miModal").style.display = "flex";
}
document.getElementById("cerrarModal").onclick = function() {
  document.getElementById("miModal").style.display = "none";
}

5. Resumen

  • El modal se oculta con display: none y se muestra con display: flex.
  • Se coloca con position: fixed para cubrir toda la pantalla.
  • Es ideal para mensajes importantes o interacciones rápidas.
