📂 Etiquetas <details> y <summary>

Estas etiquetas permiten mostrar u ocultar contenido fácilmente sin usar JavaScript. Son perfectas para secciones como preguntas frecuentes, instrucciones, contenidos extensos, etc.

  • <summary>: el texto que se ve primero (como un título)
  • <details>: contiene el contenido oculto que se expande
  • ✅ Semánticas, accesibles y con comportamiento automático

⚙️ Ejemplo interactivo

¿Qué es HTML?

HTML es el lenguaje de marcado que usamos para estructurar páginas web.

¿Qué es CSS?

CSS es el lenguaje que usamos para dar estilo a los elementos HTML.

¿Por qué usar <details>?
  • No necesita JavaScript
  • Es fácil de usar
  • Se adapta bien a contenido expandible

💻 Código fuente

<details>
  <summary>¿Qué es HTML?</summary>
  <p>HTML es el lenguaje de marcado que usamos para estructurar páginas web.</p>
</details>

<details open>
  <summary>¿Por qué usar <details>?</summary>
  <ul>
    <li>No necesita JavaScript</li>
    <li>Es fácil de usar</li>
    <li>Se adapta bien a contenido expandible</li>
  </ul>
</details>
