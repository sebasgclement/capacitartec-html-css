Selectores y display

  • .container { display: grid | inline-grid; }
  • Los grid items son los hijos directos del contenedor grid.
  • Sub-rejillas: grid-template-columns: subgrid; / grid-template-rows: subgrid; (soporte moderno).

Propiedades del contenedor

  • grid-template-columns, grid-template-rows, grid-template-areas
  • grid-auto-columns, grid-auto-rows (tamaño de pistas implícitas)
  • grid-auto-flow: row | column | dense | row dense | column dense
  • gap, row-gap, column-gap
  • Alineación del contenido de la grilla: justify-content, align-content, place-content
  • Alineación de los items en sus celdas: justify-items, align-items, place-items
  • grid (shorthand): filas/columnas/áreas + auto-flow en una sola declaración.

Propiedades del ítem

  • grid-column / grid-row (e.g. grid-column: 1 / span 2)
  • grid-area (nombre de área o shorthand: row-start / column-start / row-end / column-end)
  • justify-self, align-self, place-self

Funciones y keywords para tracks

  • fr (fracciones del espacio disponible)
  • repeat(n, …) y repeat(auto-fit | auto-fill, …)
  • minmax(min, max), fit-content(x)
  • min-content, max-content, auto
  • auto-fit vs auto-fill: ambos crean “huecos” de columnas; auto-fit colapsa columnas vacías para estirar las existentes; auto-fill mantiene las pistas aunque queden vacías.

Patrones útiles

  • Grid responsive sin media queries: grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
  • Grid implícito controlado: grid-auto-rows: 150px;, grid-auto-flow: row dense;
  • Áreas nombradas (layout semántico) con grid-template-areas.
  • Líneas nombradas: grid-template-columns: [full-start] 1fr [content-start] repeat(12, minmax(0, 70px)) [content-end] 1fr [full-end];

Snippets listos para copiar

Starter mínimo:

.grid {
  display: grid;
  gap: 12px;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

Grid áreas “Holy Grail”:

.layout {
  display: grid;
  grid-template-columns: 220px 1fr 260px;
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "footer footer footer";
  gap: 12px;
}
header{grid-area:header} nav{grid-area:nav}
main{grid-area:main} aside{grid-area:aside}
footer{grid-area:footer}

Shorthand + dense:

.masonry-like {
  display:grid;
  grid: masonry / repeat(4, 1fr); /* (nota) 'masonry' real no es estándar en grid; usar dense */
  grid-auto-flow: row dense;
  grid-auto-rows: 140px;
}

Tips y debugging

  • Activá la superposición de Grid en DevTools (Chrome/Firefox) para ver líneas y áreas.
  • Usá outline:1px dashed en items para inspeccionar rápidamente.
  • Preferí place-* como shorthands de justify-* + align-*.
  • Combiná Grid (layout) + Flex (alineación en componentes internos) cuando tenga sentido.
