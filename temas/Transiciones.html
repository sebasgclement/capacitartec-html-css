Transiciones en CSS

1) ¿Qué son?

Las transiciones permiten animar suavemente los cambios entre dos estados de una propiedad (por ejemplo, color normal → color en :hover), sin usar JavaScript.

  • transition-property: qué propiedad(s) animar (ej: transform, opacity).
  • transition-duration: cuánto dura (ej: .3s).
  • transition-timing-function: curva de aceleración (ej: ease, linear, cubic-bezier(...)).
  • transition-delay: cuándo empieza (ej: .2s).

También podés usar el shorthand transition para definir todo en una sola línea.

5) Inputs con transiciones de foco

input{
  transition: border-color .25s ease, box-shadow .25s ease;
}
input:focus{
  border-color:#0d6efd;
  box-shadow:0 0 0 4px rgba(13,110,253,.15);
}

6) Buenas prácticas

  • Animá preferentemente transform y/o opacity (más fluido; menos repaints).
  • Algunas propiedades no se pueden animar (ej: display). Alternativa: animar opacity/max-height.
  • Definí un estado inicial y final claro (normal vs :hover/.is-open).

7) Mini ejercicio

Crea “chips” interactivos que suban y cambien color al pasar el mouse (abajo ya están hechos para que te guíes):

.chip{

8) Resumen rápido

  • Usá transition para animar el pasaje entre estados.
  • Propiedades clave: property, duration, timing-function, delay.
  • Shorthand: transition: propiedad duración timing delay.
  • Mejor rendimiento con transform y opacity.
  • display no es animable: buscá alternativas (opacity, height/max-height).
