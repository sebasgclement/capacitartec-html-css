Resumen CSS Básico

🎯 Selectores

  • p → Todos los párrafos
  • .clase → Elementos con class="clase"
  • #id → Elemento con id="id"
  • div p → P dentro de un DIV

✍️ Propiedades de texto

  • color: color del texto
  • font-size: tamaño de fuente
  • font-family: tipografía
  • text-align: alineación

🧮 Unidades de medida

  • px: pixeles fijos
  • em, rem: relativas al tamaño del texto
  • %: proporción respecto al contenedor
  • vh, vw: altura o ancho del viewport

🎨 Color

  • color: red;
  • color: #ff0000;
  • color: rgb(255, 0, 0);

📦 Modelo de caja (Box model)

  • width, height
  • padding: espacio interno
  • margin: espacio externo
  • border: borde del elemento

📏 Box Sizing

Usamos box-sizing: border-box para incluir padding y border en el tamaño total del elemento.

🟦 Fondos

  • background-color
  • background-image
  • background-size, position, repeat, attachment

🌈 Gradientes

  • linear-gradient()
  • radial-gradient()

🌫️ Sombras

  • box-shadow: sombra en cajas
  • text-shadow: sombra en texto

🧪 Ejemplo combinado

div {
  color: white;
  background: linear-gradient(to right, blue, purple);
  padding: 20px;
  margin: 10px;
  border: 2px solid white;
  box-shadow: 5px 5px 15px rgba(0,0,0,0.3);
}
