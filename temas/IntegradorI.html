Integrador 1 — Inscripción Curso Frontend

Objetivo: Crear una landing page de inscripción para un curso de Frontend utilizando HTML y CSS aplicando buenas prácticas y accesibilidad.

Requerimientos

  • Usar etiquetas semánticas: <header>, <main>, <section>, <footer>.
  • Incluir un formulario con <label> + for, required y tipos de input adecuados.
  • Agregar campos: nombre, email, curso (select), comentarios (textarea) y botón de envío.
  • Aplicar estilos: tipografía, colores, botones y distribución de contenido.
  • Garantizar contraste adecuado y foco visible en elementos interactivos.

Cuando termines, compará tu resultado con el ejemplo propuesto.
