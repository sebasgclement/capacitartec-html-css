Metodología BEM

1. ¿Qué es BEM?

BEM significa Block, Element, Modifier y es una metodología para nombrar clases en CSS de forma clara y escalable.

Su objetivo es que el código sea fácil de entender, mantener y reutilizar.

2. Estructura básica

  • Bloque (Block): Componente independiente que tiene sentido por sí mismo. Ej: .menu, .card.
  • Elemento (Element): Parte interna del bloque que cumple una función. Se nombra con doble guion bajo __. Ej: .card__title.
  • Modificador (Modifier): Variante o estado del bloque o elemento. Se nombra con doble guion medio --. Ej: .card--highlight.

3. Ejemplo práctico

Producto Destacado
Este producto es el más vendido del mes.

/* CSS con BEM */
.card { ... }                /* Bloque */
.card__title { ... }         /* Elemento */
.card__description { ... }   /* Elemento */
.card--highlight { ... }     /* Modificador */

4. Ventajas de usar BEM

  • Evita conflictos de nombres.
  • Fácil de leer y entender.
  • Permite reutilizar componentes.
  • Escalable en proyectos grandes.
