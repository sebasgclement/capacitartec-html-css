Pseudoclases en CSS

1. ¿Qué son?

Las pseudoclases son palabras clave que se añaden a un selector para especificar un estado especial del elemento. No modifican el HTML, sino que aplican estilos en situaciones específicas.

Se escriben con dos puntos: selector:pseudoclase

2. Pseudoclases comunes de interacción

  • :hover → cuando el mouse pasa sobre el elemento.
  • :active → cuando el elemento está siendo clickeado.
  • :focus → cuando un campo de formulario está activo.
  • :visited → para enlaces ya visitados.

3. Pseudoclases estructurales

Sirven para seleccionar elementos según su posición en el DOM:

  • :first-child → primer hijo.
  • :last-child → último hijo.
  • :nth-child(n) → hijo número n.
  • :nth-of-type(n) → elemento número n de su tipo.

4. Negación

La pseudoclase :not() permite aplicar estilos a todos los elementos que NO cumplen una condición.

5. Resumen visual

  • Las pseudoclases definen estados o posiciones de un elemento sin modificar el HTML.
  • Se usan con : antes de la palabra clave (:hover, :first-child, etc.).
  • Tipos principales:
    • Interacción → hover, active, focus, visited.
    • Estructurales → first-child, last-child, nth-child(n).
    • Negación → not().
