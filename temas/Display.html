Propiedad display en CSS

1. ¿Qué es display?

La propiedad display define cómo se muestran los elementos en la página y cómo interactúan con el flujo del documento.

2. Valores más comunes

  • block: Ocupa todo el ancho disponible y empieza en una nueva línea. Ej: <div>, <p>.
  • inline: Ocupa solo el espacio necesario, sin saltar de línea. Ej: <span>, .
  • inline-block: Similar a inline, pero permite modificar alto y ancho.
  • flex: Activa el modelo de cajas flexibles.
  • grid: Activa el modelo de cuadrícula.
  • none: Oculta el elemento y no ocupa espacio.

4. Mini ejercicio

Crea un menú horizontal usando display: inline-block; para los elementos de la lista.


<ul class="menu">
  <li>Inicio</li>
  <li>Servicios</li>
  <li>Contacto</li>
</ul>

.menu li {
  display: inline-block;
  margin: 0 10px;
}

5. Resumen visual

  • block → Ocupa todo el ancho y empieza nueva línea.
  • inline → Solo ocupa su contenido, no salta de línea.
  • inline-block → Como inline pero con control de alto/ancho.
  • flex → Diseño flexible (1D).
  • grid → Diseño en cuadrícula (2D).
  • none → No se muestra ni ocupa espacio.
