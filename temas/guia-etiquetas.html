Guía de etiquetas HTML básicas

Esta guía contiene ejemplos y explicaciones de las etiquetas más comunes del lenguaje HTML.

<p> Párrafos

Se usa para agrupar texto en bloques. Se puede usar con atributos como style o class.
<p>Este es un párrafo de ejemplo.</p>

Este es un párrafo de ejemplo.

<h1> a <h6> Encabezados

Los encabezados definen títulos jerárquicos. <h1> es el más importante y <h6> el menos.
<h1>Título principal</h1> <h2>Subtítulo</h2>

Título principal

Subtítulo

Listas: <ul>, <ol>, <li>

  • <ul>: lista desordenada (con viñetas)
  • <ol>: lista ordenada (con números)
  • <li>: cada ítem de la lista
<ul> <li>HTML</li> <li>CSS</li> </ul>
  • HTML
  • CSS

<a> Enlaces

Los enlaces se usan con <a> y el atributo href para navegar a otras páginas o sitios.
<a href="https://www.google.com" target="_blank">Ir a Google</a>
Ir a Google

Bloque vs Línea

Bloque: ocupa todo el ancho disponible (ej: <div>, <p>)
Línea: solo lo necesario (ej: <span>, <a>)
<div>Soy un bloque</div> <span>Soy en línea</span>
Soy un bloque
Soy en línea

Rutas

  • Relativas: apuntan a un archivo en el mismo sitio (imagenes/logo.png)
  • Absolutas: URL completa (https://midominio.com/imagenes/logo.png)
<img src="imagenes/logo.png" alt="Logo">
<a href="paginas/contacto.html">Contacto</a>

Formularios

Se usa la etiqueta <form> junto con campos como input, textarea, select. Atributos comunes:
  • action: destino de los datos
  • method: "get" o "post"
<form>
  <input type="text" placeholder="Nombre">
  <button type="submit">Enviar</button>
</form>
