🧩 Etiquetas <fieldset> y <legend>

Estas etiquetas se usan para agrupar elementos relacionados dentro de un formulario, tanto visual como semánticamente.

  • <fieldset>: crea un bloque con borde para contener inputs relacionados.
  • <legend>: agrega un título descriptivo dentro del fieldset.
  • Mejora la organización, accesibilidad y claridad del formulario.

📌 Ejemplo de uso

Datos personales
Datos de contacto

💻 Código fuente

<form>
  <fieldset>
    <legend>Datos personales</legend>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" placeholder="Ej: Sebastián">

    <label for="apellido">Apellido:</label>
    <input type="text" id="apellido" placeholder="Ej: Clement">
  </fieldset>

  <fieldset>
    <legend>Datos de contacto</legend>
    <label for="email">Correo electrónico:</label>
    <input type="email" id="email" placeholder="Ej: sebas@email.com">

    <label for="tel">Teléfono:</label>
    <input type="tel" id="tel" placeholder="Ej: 3492-123456">
  </fieldset>
</form>
