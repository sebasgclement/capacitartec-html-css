Propiedad background en CSS

📚 Explicación teórica

  • background-color: Establece el color de fondo de un elemento.
  • background-image: Define una imagen como fondo (puede ser URL o gradiente).
  • background-repeat: Determina si la imagen se repite (horizontal, vertical o no se repite).
  • background-size: Controla el tamaño de la imagen (por ejemplo: cover, contain o auto).
  • background-position: Indica la posición de la imagen en el fondo (como center, top, left, etc.).
  • background-attachment: Establece si el fondo se desplaza con el contenido o queda fijo (scroll, fixed, local).
  • background (shorthand): Permite aplicar todas las propiedades anteriores en una sola línea.

Probá distintas propiedades para personalizar el fondo de este recuadro.
